start.SQL()

start.SQL() quiere decir que iniciara una base de datos sql en el script bash, proporcionando un ‘read -p’ que le permitira asignarla con un nombre a la base de datos, esta funcion puede ser muy util en algunos casos.

How Works?

  1. Se inicia un sql con ‘start.SQL()’ y se le asigna un nombre
  2. Se utiliza ‘edit.sql()’ para editar la base de datos, se le proporcionara un ‘read -p’ para introducir el nombre del SQL
  3. SqlSessions esta en construccion, actualmente no esta funcionando.

I Can create a table sql?

Si, con el edit.sql() puedes editar la tabla con VIM.

Funcionamiento y mantenimiento

Esta herramienta todavia esta en beta, puede contener errores.
